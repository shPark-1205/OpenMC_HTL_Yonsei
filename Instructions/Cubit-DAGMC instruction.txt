Coreform Cubit 프로그램을 통해 step 파일을 DAGMC 파일 (h5m)로 변환하는 방법.

1. Cubit 설치
	ㅇ 무료 평가판이나 (크랙) 설치
	ㅇ 홈페이지 가서 평가판 신청하거나 작성자에게 문의

2. step 파일 생성
	ㅇ CATIA 등의 CAD 프로그램에서 해석에 사용할 형상을 step 파일로 저장

3. Cubit에서 h5m 파일로 내보내기
	ㅇ Cubit에서 step 파일 import
	ㅇ Command line에 아래 입력 (Cubit_commands.txt 참고)
		- imprint all
		- merge all
	ㅇ 위 두 개의 명령어는 Cubit 상에서 share topology를 하는 작업임. 까먹지 말고 반드시 해야 함.
	ㅇ 단위 확인
		- 보통 CATIA에서 step 파일로 내보내면 [mm] 단위로 내보내짐.
		- Cubit은 다른 CAD 프로그램과 달리 단위를 인식하지 않음.
		- OpenMC의 기본 단위는 [cm]이기 때문에 Cubit에서 scale을 진행해야 함.
		- Command line: volume all scale 0.1 about 0 0 0
		- Scale을 하는 기준점을 따로 정해야 할 경우 0 0 0 대신 필요한 점 입력.
	ㅇ 볼륨 번호 및 이름 확인
		- CATIA에서 각 바디의 이름을 잘 지정했다면 step 파일로 불러왔을 때 바디 이름이 잘 불러와짐.
		- 필자는 내가 지정한 이름대로 step 파일에 저장이 되지 않는 오류가 있어서 다음과 같이 진행함.
		- Step 파일을 import할 때 Import All Names 옵션 해제
		- 불러온 step 파일의 볼륨 ID 확인 후 수동으로 이름 부여 (왼쪽 Model Tree 참고)
		- Command line: volume [volume ID] name "[Your_Cell_Name]"
			ex) volume 3 name "cell_Breeder"
	ㅇ 그룹 생성
		- 만약 동일 재료를 여러 부품이 공유할 경우 그룹 생성 권장
		- OpenMC에게 A라는 재료는 어떤 그룹이라는 것을 전달하는 과정
		- Command line: group "mat:[Your_Material_Name]" add volume [Your_Corresponding_Volume_List]
			ex) group "mat:He" add volume 3 4 5
			주의!!! mat: 뒤에 작성하는 material name은 OpenMC에서 부여한 name="material_name"과 정확히 일치해야 함.
	ㅇ 블록 생성
	    - OpenMC에게 하나의 바디를 인식시키는 과정
	    - Command line: block [block ID] add volume [volume ID]
	        ex) block 1 add volume 1
	        ex) block 1 add group 1
	ㅇ 블록 재료 부여
	    - OpenMC에게 블록 별 재료 지정하는 과정
	    - Command line: block [block ID] material "[material name]"
	        ex) block 1 material "tungsten"
	ㅇ 격자 생성
		- 옵션 1) Command line: surface all scheme trimesh\ mesh surface all
		- 옵션 2) 오른쪽 GUI 창: Mode-Mesh, Entity-Surface, Action-Mesh 선택 후 수동으로 격자 생성
	ㅇ 내보내기
		- File-Export 후 DAGMC 파일 확장자 선택
